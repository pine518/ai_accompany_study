# 成绩 Excel 导入模板说明

## 1. 模板文件

标准 CSV 模板：

- `templates/score_import_template.csv`

后续工程实现时，可基于该 CSV 生成 `.xlsx` 下载模板。

## 2. 字段说明

| 字段 | 中文名 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| student_no | 学号 | 是 | 学校或机构内唯一学生编号 |
| student_name | 学生姓名 | 是 | 用于人工核对 |
| class_name | 班级 | 是 | 例如：初一1班 |
| grade | 年级 | 是 | 例如：初一、初二、高一 |
| subject | 学科 | 是 | 例如：语文、数学、英语、物理、化学 |
| exam_name | 考试名称 | 是 | 例如：第一次月考、期中考试 |
| exam_date | 考试日期 | 是 | 格式：YYYY-MM-DD |
| score | 得分 | 是 | 实际得分 |
| full_score | 满分 | 是 | 该学科该次考试满分 |
| exam_type | 考试类型 | 否 | 月考、期中、期末、周测、模拟考等 |
| remark | 备注 | 否 | 可记录特殊情况 |

## 3. 导入校验规则

- 学号、姓名、班级、年级、学科、考试名称、考试日期、得分、满分必填。
- `exam_date` 必须为 `YYYY-MM-DD`。
- `score` 不能小于 0，不能大于 `full_score`。
- 同一学生、同一学科、同一考试名称、同一考试日期重复导入时，应提示覆盖或跳过。
- 导入结果需要生成批次记录，包含成功数、失败数、失败原因和操作人。

## 4. MVP 范围

一期 MVP 支持：

- 教师手动录入成绩。
- 使用模板批量导入成绩。
- 导入后生成各科成绩趋势图。
- 教师基于趋势图进行点评。
- 家长查看趋势、评价并记录奖励兑现。
